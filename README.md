# AI Financial Risk Analyzer

## Problem Statement
This project analyzes stock risk using historical market data.
It calculates volatility and categorizes the stock into risk levels.

## Features
- Accepts stock ticker as input
- Fetches real-time data
- Calculates annual volatility
- Displays risk category
- Instant output

## Tech Stack
- Python
- Streamlit
- Pandas
- NumPy
- yfinance

## How to Run Locally
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run the app:
   streamlit run app.py
